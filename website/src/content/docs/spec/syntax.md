  Defines how a conforming parser partitions UTF-8 text into an ordered
  sequence of commands and text blocks.
---

This document defines how a conforming parser partitions UTF-8 text into an
ordered sequence of commands and text blocks. It specifies line normalization,
backslash continuation, command detection, single-line and fenced multi-line
argument modes, and document partitioning. The specification is
format-agnostic: it defines observable behavior without prescribing
architecture, serialization, or host bindings.
