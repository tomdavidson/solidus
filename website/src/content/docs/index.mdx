## What is Solidus?

Slash commands appear in chat applications, developer tools, and AI agent
interfaces. Everyone implements them differently. Solidus provides the first
formal syntax specification for `/command` parsing, along with a pure Rust
engine that implements every rule.

```
/deploy production --region us-west-2
/mcp call_tool write_file ```json
{ "path": "/src/index.ts" }
```
/notify team --channel ops
```

The parser accepts any UTF-8 input and always returns a valid result.
Commands, text blocks, and warnings, neatly partitioned, every time.

## The Name

The `/` character is formally named U+002F SOLIDUS. The solidus itself is
named after a Roman gold coin introduced by Constantine I in 309 AD. It held
its value for over 700 years and became the standard unit of account across
the Byzantine Empire. The coin's abbreviation eventually evolved into the
typographic stroke we now call the forward slash.

A parser named Solidus, built to be stable, precise, and long-lasting.
That's not a metaphor. That's etymology.
